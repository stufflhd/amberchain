## Amberchain

Modern logistics dashboard built with React + Vite.

### Quick start
```bash
npm install
npm run dev
# build & preview
npm run build
npm run preview
```

### Scripts
- `dev`: start dev server
- `build`: production build
- `preview`: preview build
- `lint`: run ESLint

### Stack
React, Vite, Tailwind CSS, React Router, TanStack Query/Table, MapLibre GL, Zustand, i18next.
